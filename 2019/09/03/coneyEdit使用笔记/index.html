简介

coney edit 是一个很好的协助开发的软件尤其是数据批量处理上有很强大的功能

使用

conney edit的一些常使用的示例

字符串列表转大写（驼峰）

    setempId
    setempName
    setjob
    setsalary
    setcity
    cc.mc 4ch up // 转为大写且保持驼峰原则

一维数组 二维数组 先放入数组中然后自己组装

    private  Date  lastLogin
    cc.gc @w arra // 1获取值放入arra数组中

    this is lastLogin
    cc.gl arrb   // 2获取值放入arrb中

    $arra.2 $arra.3 = ($arra.2) objs[#1]; // $arrb
    cc.p  // 组装需要的数据并且打印

替换文本正则替换

    cc.ges /\w+/ a // 正则匹配并且将值放入数组中
    cc.rc 5pw $a  // 替换文本
    cc.rs /field\d+/ $a //正则替换文本

coneyEdit 命令说明


cc.开头
其中的字符代表的含义
列操作必须最后加-----c
行操作必须最后加-----l
字符串必须最后加-----s

g --- get
b --- before
a --- after
r --- replace
d --- delete
s --- sort

字符类型说明

简写 英文 说明
pw programWord 程序字
w word 单词
v visibleString 可视字符串
b blankString 空白字符串
q quotedString 引用字符串
dq doubleQuotedString 双引号字符串
sq singleQuotedString 单引号字符串
p punctuationString 标点字符串
d digitalString 数字字符串
i int int数字
f float 浮点数
a lowerCaseLetterString 小写字母字符串
A upperCaseLetterString 大写字母字符串
aA letterString 信笺字符串
asc ascString 字符串
nonAsc nonAscString 非字符串
cjk cjkString CJK
z cjkString CJK
ch char 字符

常用说明

 cc.gc 3w a // 获取第三列
 cc.gc 1q  // 获取第一列
 cc.dc 1v // 删除第一列
 cc.gac 1/\s+/ //获取正则表达式匹配的第一列之后的数据
 cc.gl @cjk // 获取包含cjk的行
 cc.gs @w // 获取匹配正则w的所有字符串
 cc.gc 3w -l /^String/ // 获取第三列匹配正则的值，添加行过滤条件
 cc.dc 1:_ //去除数据库字段下划线
 cc.mac 1:" ltrim // 去除"后字符串左边的空格
 cc.ms @i squo // 匹配的字符串加单引号 quo双引号
  cc.gac 1b a // 取空格后的第一个字符串 1b代表空格
 cc.p 9 '#<03>, ' 打印，必须有单引号

  private String #001;
 cc.pnn 6 //打印时候没有换行符
